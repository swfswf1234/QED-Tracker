### 官方文档镜像 wget --mirror 方案

**问题**：原有 DocScraper 使用 httpx+BS4 自爬，保存的 HTML 无 CSS/JS/字体，无法本地原样查看。

**方案**：通过 WSL 调用 wget --mirror 做全站镜像

**环境检测**：
- WSL: ✅ 已安装 (v2.7.3.0)
- WSL wget: ✅ GNU Wget 1.25.0
- 路径映射: `E:\` → `/mnt/e/` 可用
- 代理: WSL2 NAT 模式不镜像 Windows 127.0.0.1，直连可用

**代码变更**：

| 操作 | 文件 | 说明 |
|------|------|------|
| 新建 | `app/tools/wget_mirror.py` | wget 子进程封装，路径转换 `_win_to_wsl_path()` |
| 修改 | `app/tools/__init__.py` | 添加 WgetMirror 懒导入 |
| 修改 | `app/collectors/doc_scraper.py` | 替换 scrape() 为 WgetMirror，DOC_SOURCES 加 xgboost |
| 修改 | `scripts/hunt_docs.py` | page_count → file_count 适配 |

**验证结果 (xgboost)**：
```
dataset/official_docs/xgboost/en/stable/index.html
  125 HTML, 47 PNG, 9 CSS, 5 JS, 3 SVG, 字体文件
  总计 276 文件, 11.21 MB
  CSS/JS 引用已转换为本地相对路径 ✅
```
